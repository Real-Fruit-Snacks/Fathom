# uptime

> Show how long the system has been running, number of users, and load averages.
> More information: <https://manned.org/uptime>.

- Display the current time, uptime, number of logged-in users and load averages:

`uptime`

- Show only the system uptime in a human-readable format:

`uptime -p`

- Show the system uptime in a machine-readable format:

`uptime -s`

- Display version information:

`uptime -V`
