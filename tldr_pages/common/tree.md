# tree

> Display directories and files in a tree-like format.
> More information: <https://oldmanprogrammer.net/source.php?dir=projects/tree>.

- Display the tree of the current directory:

`tree`

- Display the tree of a specific directory:

`tree <path/to/directory>`

- Display the tree including hidden files:

`tree -a`

- Display only directories:

`tree -d`

- Limit the depth of recursion:

`tree -L <2>`

- Display the tree with file sizes in human-readable format:

`tree -h`

- Display the tree with full path prefix:

`tree -f`

- Display the tree and output to a file with ANSI graphics:

`tree -o <path/to/output.txt>`
