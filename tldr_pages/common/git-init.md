# git-init

> Create an empty Git repository or reinitialize an existing one.
> More information: <https://git-scm.com/docs/git-init>.

- Initialize a new Git repository in the current directory:

`git init`

- Initialize a repository in a specific directory:

`git init <path/to/directory>`

- Initialize a repository with a specific branch name:

`git init --initial-branch <branch_name>`

- Initialize a bare repository (no working directory):

`git init --bare`

- Initialize a bare repository in a specific directory:

`git init --bare <path/to/repository.git>`

- Reinitialize an existing repository (safe to run):

`git init`

- Initialize with a specific template directory:

`git init --template <path/to/template>`
