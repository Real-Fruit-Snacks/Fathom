# unlink

> Remove a link to a file from the filesystem.
> Similar to rm but only removes one file and does not accept options.
> More information: <https://www.gnu.org/software/coreutils/unlink>.

- Remove the specified file if it is the last link:

`unlink <path/to/file>`

- Remove a symbolic link:

`unlink <path/to/symbolic_link>`

- Remove a hard link:

`unlink <path/to/hard_link>`
