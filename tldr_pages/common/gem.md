# gem

> Ruby package manager.
> Install and manage Ruby libraries and applications (gems).
> More information: <https://guides.rubygems.org>.

- Install a gem:

`gem install <gem_name>`

- Install a specific version of a gem:

`gem install <gem_name> --version <version>`

- Update a gem to the latest version:

`gem update <gem_name>`

- Update all installed gems:

`gem update`

- Uninstall a gem:

`gem uninstall <gem_name>`

- List all installed gems:

`gem list`

- Search for a gem by name:

`gem search <search_string>`

- Show information about an installed gem:

`gem info <gem_name>`
