# node

> JavaScript runtime built on Chrome's V8 engine.
> More information: <https://nodejs.org>.

- Start an interactive Node.js shell (REPL):

`node`

- Execute a JavaScript file:

`node <path/to/script.js>`

- Execute a JavaScript file with arguments:

`node <path/to/script.js> <arg1> <arg2> [arg3...]`

- Evaluate JavaScript code from the command line:

`node -e "<code>"`

- Check syntax of a JavaScript file without executing it:

`node --check <path/to/script.js>`

- Show the installed Node.js version:

`node --version`

- Enable experimental ES modules:

`node --experimental-modules <path/to/script.mjs>`

- Inspect and debug a script using Chrome DevTools:

`node --inspect-brk <path/to/script.js>`
