# python3

> Python 3 language interpreter.
> More information: <https://docs.python.org/3/using/cmdline.html>.

- Start an interactive Python 3 shell (REPL):

`python3`

- Execute a Python 3 script:

`python3 <path/to/script.py>`

- Execute a Python 3 script with arguments:

`python3 <path/to/script.py> <arg1> <arg2> [arg3...]`

- Execute a Python 3 module:

`python3 -m <module_name>`

- Execute a Python 3 statement:

`python3 -c "<statement>"`

- Start an interactive shell after executing a script:

`python3 -i <path/to/script.py>`

- Create a virtual environment in the specified directory:

`python3 -m venv <path/to/venv>`

- Install pip packages in a virtual environment:

`python3 -m pip install <package_name>`
