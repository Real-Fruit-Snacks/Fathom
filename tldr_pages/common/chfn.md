# chfn

> Change a user's full name and other information.
> More information: <https://man7.org/linux/man-pages/man1/chfn.1.html>.

- Change the full name for the current user interactively:

`chfn`

- Change the full name for the current user:

`chfn <-f> "<full_name>"`

- Change the full name for a specific user:

`chfn <-f> "<full_name>" <username>`

- Change the room number for a specific user:

`chfn <-r> "<room_number>" <username>`

- Change the work phone for a specific user:

`chfn <-w> "<work_phone>" <username>`

- Change the home phone for a specific user:

`chfn <-h> "<home_phone>" <username>`
