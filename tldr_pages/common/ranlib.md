# ranlib

> Generate index to an archive.
> More information: <https://sourceware.org/binutils/docs/binutils/ranlib.html>.

- Generate or update the symbol index of an archive:

`ranlib <path/to/archive.a>`

- Display the version:

`ranlib <-V>`

- Update the timestamp of the symbol map:

`ranlib <-t> <path/to/archive.a>`
