# tty

> Display the filename of the terminal connected to standard input.
> More information: <https://www.gnu.org/software/coreutils/tty>.

- Display the filename of the terminal:

`tty`

- Display only the filename, without extra messages:

`tty <-s|--silent>`

- Display the filename or print nothing if standard input is not a terminal:

`tty <-q|--quiet>`
