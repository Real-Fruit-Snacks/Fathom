# ranger

> Console file manager with VI key bindings.
> More information: <https://ranger.github.io/>.

- Start ranger in the current directory:

`ranger`

- Start ranger in a specific directory:

`ranger <path/to/directory>`

- Show hidden files:

`zh`

- Select a file and copy:

`yy`

- Select a file and cut:

`dd`

- Paste copied/cut files:

`pp`

- Delete selected files:

`dD`

- Quit ranger and change to the current directory:

`S`
