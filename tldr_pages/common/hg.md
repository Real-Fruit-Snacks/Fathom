# hg

> Mercurial distributed version control system.
> More information: <https://www.mercurial-scm.org/>.

- Initialize a new repository:

`hg init`

- Clone a repository:

`hg clone <https://repo.url/path> [local_directory]`

- Add files to be tracked:

`hg add <path/to/file>`

- Commit changes with a message:

`hg commit -m "<commit_message>"`

- Show the status of the working directory:

`hg status`

- Show the commit history log:

`hg log`

- Show differences between working directory and last commit:

`hg diff [path/to/file]`

- Pull changes from a remote repository and update:

`hg pull -u`
