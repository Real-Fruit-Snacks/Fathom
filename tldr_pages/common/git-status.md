# git-status

> Show the working tree status.
> More information: <https://git-scm.com/docs/git-status>.

- Show the status of changed files:

`git status`

- Show short format status:

`git status [-s|--short]`

- Show status with branch information:

`git status [-b|--branch]`

- Show status in short format with branch info:

`git status -sb`

- Show status including untracked files:

`git status [-u|--untracked-files]`

- Show only tracked files:

`git status -uno`

- Show status ignoring submodules:

`git status --ignore-submodules`

- Show status in machine-readable format:

`git status --porcelain`

- Show verbose status with staged diff:

`git status [-v|--verbose]`
