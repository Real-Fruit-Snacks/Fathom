# most

> A pager similar to more and less, but capable of viewing multiple files in split windows.
> More information: <https://www.jedsoft.org/most/>.

- Open a file for interactive reading:

`most <path/to/file>`

- Open multiple files in split windows:

`most <path/to/file1> <path/to/file2>`

- Page down or up:

`<<Space>|<b>>`

- Search forward for a string (press `<n>` to find next match):

`</><pattern>`

- Split the screen horizontally and open another file:

`<Ctrl-x> 2`

- Switch between open windows:

`<Ctrl-x> o`

- Close the current window:

`<Ctrl-x> 0`

- Exit most:

`<q>`
