# talk

> Talk to another user.
> More information: <https://man7.org/linux/man-pages/man1/talk.1.html>.

- Request a talk session with a user on the same machine:

`talk <username>`

- Request a talk session with a user on a different machine:

`talk <username@hostname>`

- Request a talk session with a user on a specific terminal:

`talk <username> <ttyname>`
