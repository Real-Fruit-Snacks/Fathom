# uuidgen

> Generate new UUID (Universally Unique Identifier).
> See also `uuid`.
> More information: <https://man7.org/linux/man-pages/man1/uuidgen.1.html>.

- Generate a random UUIDv4:

`uuidgen <-r|--random>`

- Generate a UUIDv1 based on the current time:

`uuidgen <-t|--time>`

- Generate multiple UUIDs:

`uuidgen <-r> <-n> <5>`
