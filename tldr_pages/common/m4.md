# m4

> Macro processor. Mostly used for preprocessing other programs.
> More information: <https://www.gnu.org/software/m4/manual/m4.html>.

- Process macros in a file:

`m4 <path/to/file>`

- Define a macro before processing files:

`m4 <-D><macro_name>=<value> <path/to/file>`

- Undefine a built-in macro before processing files:

`m4 <-U><macro_name> <path/to/file>`

- Include a file before processing files:

`m4 <-I> <path/to/directory> <path/to/file>`

- Process a file with GNU M4 extensions disabled:

`m4 <-G> <path/to/file>`

- Process stdin:

`<command> | m4`
