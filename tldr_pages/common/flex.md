# flex

> Fast Lexical Analyzer generator. Often used with Bison.
> More information: <https://github.com/westes/flex>.

- Generate an analyzer from a flex file:

`flex <analyzer.l>`

- Specify the output file:

`flex <--outfile=<<analyzer.c> <analyzer.l>`

- Compile a C file generated by flex:

`cc <lex.yy.c> <-lfl>`

- Generate an analyzer that includes debug information:

`flex <--debug> <analyzer.l>`

- Specify the name prefix for generated functions (instead of `yy`):

`flex <--prefix=<<prefix> <analyzer.l>`

- Generate a scanner that does not depend on the flex library:

`flex <--noyywrap> <analyzer.l>`
