# python

> Python language interpreter.
> More information: <https://docs.python.org/3/using/cmdline.html>.

- Start an interactive Python shell (REPL):

`python`

- Execute a Python script:

`python <path/to/script.py>`

- Execute a Python script with arguments:

`python <path/to/script.py> <arg1> <arg2> [arg3...]`

- Execute a Python module:

`python -m <module_name>`

- Execute a Python statement:

`python -c "<statement>"`

- Start an interactive shell after executing a script:

`python -i <path/to/script.py>`

- Show the installed Python version:

`python --version`

- Start a simple HTTP server on port 8000:

`python -m http.server [port]`
