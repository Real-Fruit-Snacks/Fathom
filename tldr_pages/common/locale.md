# locale

> Display locale-specific information.
> More information: <https://man7.org/linux/man-pages/man1/locale.1.html>.

- Display the current locale settings:

`locale`

- Display all available locales:

`locale <-a|--all-locales>`

- Display all available character maps:

`locale <-m|--charmaps>`

- Display the value of a specific locale category:

`locale <LC_TIME>`

- Display all locale categories and their values:

`locale <-k> <LC_ALL>`
