# git-pull

> Fetch from and integrate with another repository or local branch.
> More information: <https://git-scm.com/docs/git-pull>.

- Pull changes from the default remote and branch:

`git pull`

- Pull changes from a specific remote and branch:

`git pull <remote_name> <branch_name>`

- Pull and rebase instead of merge:

`git pull --rebase`

- Pull and automatically stash/unstash local changes:

`git pull --autostash`

- Pull without committing the merge:

`git pull --no-commit`

- Pull using fast-forward only:

`git pull --ff-only`

- Pull all tags from the remote:

`git pull --tags`

- Pull changes from all remotes:

`git pull --all`
