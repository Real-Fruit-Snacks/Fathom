# tig

> Text-mode interface for Git.
> An ncurses-based git repository browser.
> More information: <https://jonas.github.io/tig/>.

- Start tig in the current repository:

`tig`

- View the commit history of a specific branch:

`tig <branch_name>`

- View the history of a specific file:

`tig <path/to/file>`

- View the blame for a file:

`tig blame <path/to/file>`

- View the stash:

`tig stash`

- View all references (branches, tags):

`tig refs`

- View the status of the working tree:

`tig status`

- View commits in a specific range:

`tig <commit1>..<commit2>`
