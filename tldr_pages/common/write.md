# write

> Write a message on a specific user's terminal.
> More information: <https://man7.org/linux/man-pages/man1/write.1.html>.

- Send a message to a specific user (end with Ctrl+C):

`write <username>`

- Send a message to a specific user on a specific terminal:

`write <username> <ttyname>`

- Send a message from a file:

`write <username> << path/to/file>`
