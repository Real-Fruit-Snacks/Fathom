# sleep

> Delay for a specified amount of time.
> More information: <https://www.gnu.org/software/coreutils/sleep>.

- Delay in seconds:

`sleep <1.5>`

- Delay in minutes:

`sleep <2m>`

- Delay in hours:

`sleep <4h>`

- Delay in days:

`sleep <7d>`

- Execute a command after a delay:

`sleep <30> && <command>`
