# id

> Display user and group identity information.
> More information: <https://man7.org/linux/man-pages/man1/id.1.html>.

- Display current user's UID, GID, and groups:

`id`

- Display identity information for a specific user:

`id <username>`

- Display only the effective user ID:

`id -u`

- Display only the effective group ID:

`id -g`

- Display all group IDs:

`id -G`

- Display user name instead of ID number:

`id -un`

- Display group name instead of ID number:

`id -gn`

- Display all group names instead of IDs:

`id -Gn`
