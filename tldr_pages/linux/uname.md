# uname

> Print system information about the current machine and operating system.
> More information: <https://man7.org/linux/man-pages/man1/uname.1.html>.

- Print all system information:

`uname -a`

- Print the kernel name:

`uname -s`

- Print the kernel release version:

`uname -r`

- Print the kernel version:

`uname -v`

- Print the machine hardware name (architecture):

`uname -m`

- Print the network hostname:

`uname -n`

- Print the operating system name:

`uname -o`

- Print the processor type:

`uname -p`
