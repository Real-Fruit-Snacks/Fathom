# w

> Display who is logged in and what they are doing.
> More information: <https://man7.org/linux/man-pages/man1/w.1.html>.

- Display information about all logged-in users:

`w`

- Display information for a specific user:

`w <username>`

- Display without the header line:

`w -h`

- Display short format (without login time, JCPU, PCPU):

`w -s`

- Display FROM field (remote hostname):

`w -f`

- Display IP addresses instead of hostnames:

`w -i`

- Show old style output (blank for local users):

`w -o`

- Suppress header and use short format:

`w -hs`
