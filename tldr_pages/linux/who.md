# who

> Display who is currently logged in.
> More information: <https://man7.org/linux/man-pages/man1/who.1.html>.

- Display all logged-in users:

`who`

- Display the current user only:

`who am i`

- Display login time and idle time:

`who -u`

- Display column headers:

`who -H`

- Display all available information:

`who -a`

- Show system boot time:

`who -b`

- Display users and their message status:

`who -T`

- Count logged-in users:

`who -q`
